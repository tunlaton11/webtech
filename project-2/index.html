Vue.js

มาทำความรู้จักกับ Vue.js

Vue.js คืออะไร ? Vue.js เป็น Front-End Framework ที่เป็น Open-source และเขียนด้วยภาษา Javascript มีลักษณะเป็น single-page applications กำลังเป็นที่นิยมในปัจจุบัน ใช้งานและเรียนรู้ง่าย และกินทรัพยากรน้อยมาก

ข้อดีและข้อเสียของ Vue.js

ข้อดี ข้อเสีย
ใช้งานง่าย เรียนรู้ได้ไว ไม่เหมาะสำหรับเว็บไซต์ที่มีขนาดใหญ่มาก
กินพื้นที่และทรัพยาการน้อย เพียงแค่ 20KB ยังไม่มี Community ที่ใหญ่พอ
สามารถจัดการ DOM ได้ง่ายดาย ไม่เสถียรเท่าที่ควรเพราะเป็น Framework ใหม่
ทำ Unit-testing ได้ง่ายและรวดเร็ว ยังไม่มี Mobile Framework

ทำไมถึงควรใช้ Vue.js ?

Feature หลักและคุณสมบัติของ Vue.js

Virtual DOM

คือการสร้าง Tree-Object ซึ่งคือตัวแทนของ Real DOM หมายความว่า แทนที่เราจะสร้าง div จริงๆไว้ใน Browser เราก็สร้างมันไปเก็บไว้ใน Memory แทน แทนที่จะไปจัดการกับ Real DOM ด้วยการใช้ DOM API แบบเก่า ๆ ก็ใช้วิธีการ patch จากนั้นก็ Render UI ส่วนที่ถูกกระทำใหม่ และแทนที่จะต้องไปไล่ DOM ว่าอยู่ส่วนไหน ก็ใช้การ ref มาทำแทน โดย Virtual DOM ของ Vue.JS จะใช้ของ Snabbdom โดยมีการการันตรีมาแล้วว่า เร็วขึ้นและลดการใช้หน่วยความจำลง 2–4 เท่าตัว

Two-Way Data Binding

Vue.js มี two-way binding feature จากสถาปัตยกรรม MVVM ซึ่งจะช่วยจัดการหรือกำหนดค่าของ HTML attributes ซึ่งทำให้มีประสิทธิภาพเร็วขึ้น two-way binding feature ของ Vue.js จะมั่นใจได้ว่าการเปลี่ยนแปลงในหน้า UI ข้อมูลจะถูกส่งไปอย่างรวดเร็วและแสดงผลออกมาทันที ซึ่งต่างจาก React.js ที่เป็น one-way

Templates

Vue.js ใช้ HTML-based DOM ซึ่งรวมกับข้อมูลของ Vue.js เพื่อที่จะแปลงไปเป็น templates ภายใต้ Virtual DOM Render functions ซึ่ง Developer สามารถกำหนด templates โดยใช้ render function ซึ่ง templates จะมีลักษณะเป็น single root node แต่ใน Vue.js 3.0 สามารถใช้เป็น multiple root node ได้แล้ว

Easy to Understand

นี่เป็หนึ่งเหตุผลที่ว่าทำไม Vue.js จึงได้รับความนิยม เพราะว่ามีโครงสร้างที่ตรงไปตรงมา มีการออกแบบที่โดดเด่น เพื่อใช้ในการจัดเก็บข้อมูล, การทำ life cycle และอิสระในการกำหนดโครงสร้างเอง นอกจากนี้ยังมีคุณลักษณะพิเศษเช่น viewers, registered properties และ directives มีส่วนให้ cycle ในการพัฒนาเป็นเรื่องที่ง่ายและทันสมัย ช่วยทำให้องค์กรประหยัดเวลาได้มาก

Computed Properties

นี่เป็นหนึ่ง feature ที่สำคัญของ Vue.js ที่ช่วยในการจัดการกับ UI Element และเพิ่มลูกเล่นที่มี logic ซึ่งไม่จำเป็นต้องเข้ารหัสเลย ดังนั้นเราสามารถนำ Computed property มาใช้ได้เพื่อแยกส่วนการคำนวณออกไปจากส่วนการรับข้อมูลจากผู้ใช้งานโดยที่ Computed property จะคอยตรวจสอบการเปลี่ยนแปลงให้เราเองจากนั้นจะทำการเรียกหรือ trigger การทำงานให้เองแบบอัตโนมัติหรือเราเรียก Computed property ว่า getter method


        export default {
            data () {
                text: 'hello world'
            },
            computed: {                      // computed property
                formattedText: function () {
                    return this.text.toUpperCase()
            }
        }
    }

Vue.js เบื้องต้น

วิธีการใช้งาน Vue.js แบบเบื้องต้นที่ใครๆเริ่มทำได้

การเริ่มต้นใช้งาน

การเริ่มต้นใช้งาน Vue.js จะมีแนวทางได้ 2 วิธีดังนี้

  1. 1. CDN
    2.                         <script src="https://cdn.jsdelivr.net/npm/vue">test</script>
  2. 2. CLI * จะแนะนำวิธีนี้เนื่องจากเป็นที่นิยม และสร้าง Projectได้อย่างรวดเร็วผ่าน npm รวมถึงมี tools ให้หลังจากการติดตั้ง
    3.                         $ npm install -g @vue/cli

มาเริ่มสร้าง Project

หลังจากเรา run คำสั่ง

                    $ npm install -g @vue/cli

เราจะสามารถใช้งาน Vue.js CLI ได้แล้ว จากนั้นในการสร้าง project เราจะใช้คำสั่ง

                    $ vue create my-app     //my-app คือชื่อโปรเจคของเรา

ซึ่งจะมีหน้าของ CLI มาให้เลือก preset ซึ่งจะแนะนำให้เลือก Vue 3 เพราะเป็นเวอร์ชั่นใหม่


เมื่อทำการสร้าง project เรียบร้อยแล้ว cd ไปที่ folder ของโปรเจค จากนั้นจะทำการ run ทดสอบหน้า project ของเราด้วยคำสั่งดังนี้

                    $ npm run serve

จากนั้นเข้า browser ไปตาม address ที่ปรากฎ จะได้หน้าตาดังนี้ นั่นหมายถึง Vue.js สามารถใช้งานได้แล้ว
